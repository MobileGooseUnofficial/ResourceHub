### If your mod isn't working, refer to this checklist to see what step might be missing:

**Checklist:**

- [ ] Latest version of Mobile Goose
- [ ] Mod is installed in Cydia/Zebra/Sileo
- [ ] Mod is enabled under Mods
- [ ] Disable the goose
- [ ] Restart SpringBoard
- [ ] Enable the goose
